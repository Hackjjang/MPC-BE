# Media Player Classic - Black Edition (MPC-BE)
---

MPC-BE는 윈도용 범용 미디어 플레이어입니다.
이 프로젝트는 "미디어 플레이어 클래식"(Gabest) 및 "미디어 플레이어 클래식 - 홈 시네마"(Casimir666)의 소스 코드를 기반으로 독립적으로 개발되었습니다.

## 시스템 요구 사항 :
* SSE2를 지원하는 프로세서
* 윈도 7, 8, 8.1, 10, 11 32비트/64비트
